Feiertage API

Gesetzliche Feiertage in Deutschland & Osterreich

Public API No Auth DE + AT

About

This API provides all public holidays (gesetzliche Feiertage) for all 16 German federal states (Bundeslander) and all 9 Austrian states, plus special days like Easter, Advent Sundays, Thanksgiving, and more.

The API is completely open — no authentication, no API keys, no rate limits. Just use it.

API Endpoints

All Regions

GET /api/regions?year=2026

Returns all regions with their complete holiday lists for a year.

Optional: ?country=de or ?country=at to filter.

Optional: ?inkl_sonntage=true to include Sundays.

Output formats: ?format=xml|csv|tsv|txt

Single Region

GET /api/region/Bayern?year=2026

Returns holidays for a specific region. Use region name (e.g. "Bayern", "Berlin", "Wien").

Is Holiday?

GET /api/isFeiertag?date=2026-11-15&region=Niederosterreich

Check if a date is a holiday, optionally filtered by region.

Holidays for a Year

GET /api/feiertage?year=2026&region=Bayern

Flat list of holidays. region parameter is optional.

Holidays by Date

GET /api/feiertage/2026-12-25

Check which holidays occur on a specific date.

Easter Calculation

GET /api/easter?year=2026

Get the Easter date for any year (Gauss algorithm).

Available Regions

Germany

Baden-Wurttemberg, Bayern, Berlin, Brandenburg, Bremen, Hamburg, Hessen, Mecklenburg-Vorpommern, Niedersachsen, Nordrhein-Westfalen, Rheinland-Pfalz, Saarland, Sachsen, Sachsen-Anhalt, Schleswig-Holstein, Thuringen, Deutschland

Austria

Burgenland, Karnten, Niederosterreich, Oberosterreich, Salzburg, Steiermark, Tirol, Vorarlberg, Wien, Osterreich
