Feiertage API

Gesetzliche Feiertage in allen deutschen und osterreichischen Bundeslandern — offen, kostenlos, ohne Auth.

Public API No Auth DE + AT JSON / XML / CSV / TSV / TXT
Endpoints

REST API

Alle Endpunkte akzeptieren ?format=json|xml|csv|tsv|txt

GET /api/regions?year=2026 Alle Regionen & Feiertage
GET /api/region/{name}?year=2026 Eine Region
GET /api/feiertage?year=2026 Alle Feiertage eines Jahres
GET /api/feiertage/2026-12-25 Feiertage an einem Datum
GET /api/isFeiertag?date=2026-11-15&region=Niederoesterreich Ist ein Datum ein Feiertag?
GET /api/easter?year=2026 Osterdatum (Gauss)
